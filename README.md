# E-Commerce Sales Data Analysis (SQL)

## Overview
This project analyzes e-commerce sales data using SQL to extract insights on revenue, customers, and profitability.

## Tools Used
- SQL (SQLite)

## Dataset
This project uses the Superstore dataset.
Source:
[View Dataset] ([https://www.kaggle.com/code/rohitsahoo/eda-superstore-dataset])
Note: Dataset not uploaded due to size. Please download from the link above.

## How to Use Dataset
1. Download the dataset from the link above  
2. Import into SQLite or your preferred database  
3. Run queries from queries.sql  

## Key Analysis
- Total sales and profit
- Sales by region
- Top customers
- Profit by category
- Monthly trends
- Loss-making products

## Key Insights
- Identified top-performing regions
- Found high-value customers
- Detected products generating losses

## Outcome
This project demonstrates SQL skills in data aggregation, grouping, and business analysis.
